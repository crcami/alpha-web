# Alpha Steel Web

Sistema de gestão para Empresa de Materiais Industriais desenvolvido em React + TypeScript.

## Início Rápido

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar testes E2E
npm run test:e2e

# Abrir Cypress interativo
npm run test:e2e:open
```

## Funcionalidades

- ✅ Gestão de Produtos e Matérias-Primas
- ✅ Planejamento de Produção
- ✅ Gerenciamento de Unidades de Medida
- ✅ Autenticação com JWT e Refresh Token
- ✅ Tema Claro/Escuro
- ✅ Sistema de Notificações
- ✅ Perfil de Usuário

## Configuração

Crie um arquivo `.env` na raiz do projeto:

```env
VITE_API_BASE_URL=http://localhost:8080/api/v1
```
